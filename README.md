# StudentCourse
1) Developed a full-stack web application using HTML, CSS, JavaScript, Bootstrap, Java, Thymeleaf, Spring Boot, Spring
Data JPA, Spring Boot Validation, MySQL, and Lombok.

3) Built a robust Admin Panel to manage courses and course content, including adding, updating, and deleting courses.
Managed course images with Cloudinary, handled admin profile images stored locally with update, view, and delete
options, and managed course-wise user feedback with view, update, and delete capabilities. Implemented secure
authentication for admin login, forgot password, and logout.

5) Designed a comprehensive User Panel with registration and authentication. Users can view courses after login, purchase
one course at a time, access course content after purchase, and add, update, or delete feedback only for the courses
they have purchased.
