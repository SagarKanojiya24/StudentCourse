# StudentCourse

• Developed a full-stack web application using HTML, CSS, JavaScript, Bootstrap, Java, Thymeleaf, Spring Boot, Spring
Data JPA, Spring Boot Validation, MySQL, and Lombok.

• Built an Admin Panel to manage courses (add, update, delete), handle authentication (login, forgot password, logout),
upload course images via Cloudinary, store admin profile images locally, and view course-wise user feedback.

• Designed a User Panel with registration and authentication, course browsing, and feedback management, allowing users
to add, update, or delete one feedback per course, and storing user profile images locally.
